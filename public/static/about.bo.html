བུདྡྷ་དྲ་ཐོག་དཔེ་མཛོད་ཀྱི་ངོ་སྤྲོད་མདོར་བསྡུས།

BDRC ཚོགས་པས་ཙི་ཅང་སློབ་གྲྭ་ཆེན་མོའི་ནང་བསྟན་དཔེ་ཚོགས་དང་ཞིབ་འཇུག་ལྟེ་གནས་བཅས་ལ་མཉམ་འབྲེལ་བྱས་ཏེ། སྤྱིར་ཤེས་ཡོན་གྱི་སྡེ་ཚན་དང་ནང་པའི་ཚོགས་སྡེ་ཁག་དེ་བཞིན་ནང་པའི་གསུང་རབ་ལ་དོན་གཉེར་ཆེ་བའི་མང་ཚོགས་བཅས་ཀྱི་དགོས་མཁོར་དམིགས་ནས་ནང་པའི་གསུང་རབ་རྣམས་ཐུན་མོང་གིས་ལོངས་སྤྱོད་བྱ་སའི་བུདྡྷ་དྲ་ཐོག་དཔེ་མཛོད་ (BUDA) ཟེར་བ་ཞིག་གསར་བསྐྲུན་བྱས། འདིའི་ནང་དཔེ་ཀློག་ལ་ཉེ་བར་མཁོ་བའི་ཁྱད་ཆོས་དང་ཐབས་ལམ་གསར་པ་མང་པོ་བསྣན་ཡོད། འདི་བརྒྱུད་ནས་རྒྱ་སྐད་དང་ལེགས་སྦྱར་སྐད། བོད་སྐད། པཱ་ལི། འབར་མའི་སྐད། ཁཱ་མེར་སྐད་བཅས་ཀྱི་ནང་བྲིས་པའི་ནང་པའི་ཆོས་གཞུང་གཏོས་ཆེ་བ་ཞིག་བལྟ་ཀློག་བྱ་ཐུབ་པ་ཡིན། BUDAདྲ་ཐོག་དཔེ་མཛོད་ནི་སྤྱི་སྤྱོད་དང་གང་ཐུབ་ཀྱིས་ཆོག་མཆན་མི་དགོས་པར་ཐུན་མོང་གིས་བལྟ་ཀློག་བྱ་ཐུབ་པར་འབད་བརྩོན་བྱས་ཡོད། འདིའི་ནང་གི་དཔེ་ཚོགས་རྣམས་ཀྱང་BDRC ཚོགས་པ་དང་མཉམ་འབྲེལ་ཚོགས་པ་རྣམས་ཀྱིས་མཁོ་སྤྲོད་བྱས། Robert H. N. ཧོ་ནང་མའི་ཐེབས་རྩ་ཁང་གི་གཏོང་ཕོད་ལ་བརྟེན་ནས་དཔེ་མཛོད་འདི་གྲུབ་པ་ཞིག་ཡིན།

BUDA དྲ་ཐོག་དཔེ་མཛོད་ཀྱིས་སྤྱོད་མཁན་གྱི་དགོས་མཁོ་ལྟར་སྐད་རིགས་སྣ་ཚོགས་ནང་འཚོལ་ཞིབ་དང་སྤྱོད་མཁན་ལ་ལམ་སྟོན་བྱེད་པའི་དྲི་བ་དྲི་ལན། དེ་བཞིན་BDRC ཡི་ཡིག་ཚོགས་ནས་རང་ཉིད་ཀྱི་མ་གཞིའི་གཞི་གྲངས་སྒྲིག་འགོད་ཐད་ཀར་ཕབ་ལེན་བྱེད་པ་བཅས་ཀྱི་ཞིབ་ཕྲའི་གནས་གང་མང་ཞིག་མཁོ་སྤྲོད་བྱས་ཡོད། ད་དུང་དྲ་ཐོག་དཔེ་མཛོད་འདིས་ནང་པའི་གསུང་རབརྣམས་ལ་ཞིབ་ཕྲའི་ཞིབ་འཇུག་དང་བཀོལ་སྤྱོད་ཟབ་མོའི་ཡོང་སླད་སྤྱོ་སྒོ་ཆེ་བའི་ལག་ཆ་སྣ་ཚོགས་ཤིག་ཀྱང་བསྐྲུན་ཡོད། དྲ་ཐོག་དཔེ་མཛོད་འདིའི་རྩ་བའི་སྒྲོམ་གཞི་ནི་སྤྱིའི་ཚད་གཞི་བཀོལ་སྤྱོད་བྱས་པའི་སྤྱི་སྤྱོད་ཅིག་ཡིན་ཞིང་། རིན་མེད་ངང་ལོངས་སུ་སྤྱོད་ཆོག འདིའི་རྨང་གཞིའི་ཡིག་ཆ་རྣམས་དང་འཇུག་སྤྲོད་བྱེད་པའི་བཀོལ་སྤྱོད་ལམ་སྟོན་དང་བཅས་པ་ང་ཚོའི་གྷི་ཐབ་དྲ་ཚིགས་ GitHub page ནང་བཞག་ཡོད།

བུདྡྷ་དྲ་ཐོག་དཔེ་མཛོད་ཀྱི་བྱེད་ནུས་འདོན་སྤྲོད་ཀྱི་རྩ་བའི་ཐབས་རྩལ་རིགས་གཉིས་བཞག་ཡོད་པ་ནི་ International Image Interoperability Framework (IIIF) དང་ Linked Open Data ཟེར་བ་འདི་གཉིས་ཡིན།

IIIF ནི་དྲ་ལམ་གྱི་བཤེར་པར་རྣམས་འཆར་སྟོན་དང་བརྒྱུད་སྐུར་བྱེད་པའི་ལག་རྩལ་གྱི་གང་བུ་ཞིག་ཡིན་ཞིང་། IIIF ནི་གཙོར་ཆེར་འཛམ་གླིག་གི་དཔེ་མཛོད་ཁང་རྣམས་དང་འགྲེམ་སྟོན་ཁང་། ཞིབ་འཇུག་སློབ་གླིང་ཁག་རྣམས་ཀྱིས་བཀོལ་སྤྱོད་བྱེད་པ་ཡིན། རྒྱུ་མཚན་ནི་ IIIF Consortium ཡི་དྲ་ཚིགས་ནང་གསལ་བརྗོད་བྱས་པ་ལྟར། འདི་ནི་བཤེར་པར་རམ་འདྲ་པར་རིགས་ཀྱི་གཞི་རྩ་བྱས་པའི་ཡིག་ཆ་རྣམས་བལྟ་ཀློག་གམ་བཀོལ་སྤྱོད་ཞིབ་ཕྲ་བྱེད་པར་ཆེད་དུ་དམིགས་ནས་ཇུས་འགོད་བྱས་པ་ཞིག་རེད། དེའི་ཕྱིར་ BUDA དཔེ་མཛོད་ཀྱིས་ཀྱང་ང་ཚོའི་གཏོས་ཆེ་བའི་བཤེར་པར་རྣམས་འཆར་སྟོན་དང་བཀོལ་སྤྱོད་སྤུས་ལེགས་ཡོང་བར་ IIIF འདི་བཀོལ་སྤྱོད་བྱེད་བཞིན་ཡོད།

IIIF འདི་བཀོལ་སྤྱོད་བྱེད་པའི་དགོས་དོན་གཞན་ཞིག་ནི་འདི་ལྟར་ཏེ། BDRC ཚོགས་པས་ནང་པའི་ཆོས་གཞུང་རྣམས་ཉར་སྲུང་བྱ་ཕྱིར་གསར་བསྐྲུན་བྱས་པའི་ BUDA དྲ་ཐོག་དཔེ་མཛོད་ཀྱང་Linked Open Data གཞིར་བྱས་ཏེ་བཟོས་ཡོད། Linked Open Data ཟེར་བ་ནི་དྲ་ཚིགས་ནང་གི་གནས་ཚུལ་རྣམས་ཀྱི་སྒྲོམ་གཞི་འཛིན་པའི་ཐབས་ལམ་ཞིག་དང་། འདིས་ད་དུང་དྲ་ཚིགས་ནང་གི་གཞི་གྲངས་རྣམས་ཀློག་ཐུབ་པའི་ཡིག་གཟུགས་ཀྱི་རྣམ་པ་རུ་འཆར་སྟོན་ཐུབ་པ་ཞིག་བཟོ་བ་ཡིན། འདིའི་དོན་ནི་མིའི་རམ་འདེགས་དེ་ཙམ་མི་དགོས་པར་གློག་ཀླདརང་གིས་དེ་དག་གང་ཡིན་ངོས་འཛིན་དང་བརྒྱུད་སྤྲོད་བྱེད་ཐུབ་པ། དེ་བཞིན་གནས་ཚུལ་རྣམས་བཀོལ་སྤྱོད་བྱེད་ཐུབ་པ་ཞིག་བཟོ་བ་རེད། གཞི་གྲངས་གསོག་མཛོད་ཀྱི་ཚན་པ་འདི་དག་ནི་ལུང་འདྲེན་དང་དྲི་བ་འདྲི་སླ་བ། དྲ་ཐག ཁ་གསབ་བཅས་བྱེད་སླ་བ་ཡིན། ང་ཚོས་ད་དུང་ Linked Open Data ཡི་ཐབས་ལམ་བཀོལ་ཏེ་གཞི་གྲངས་མཛོད་ཉར་ཚགས་བྱེད་ཐུབ་པ་ཡིན། ཐབས་ལམ་འདི་བརྒྱུད་ནས་དེ་དག་ཡིག་ཆ་ཧྲིལ་བོ་གཅིག་གི་ངོ་བོར་ཉར་ཐུབ་པ་ཡིན། དེའི་ཕྱིར་ཡིག་ཆ་དེ་དག་དུས་ཡུན་རིང་བོར་ཉར་སྲུང་བྱེད་སྟབས་བདེ་རུ་གཏོང་འགྲོ། ཨང་བསྒྱུར་བྱེད་པའི་མི་ཆོས་ལས་གཞི་དང་དཔེ་མཛོད་ཁང་མི་ཉུང་བ་ཞིག་གིས་Linked Open Data བཀོལ་སྤྱོད་བྱེད་བཞིན་ཡོད། རྒྱུ་མཚན་ནི་དེས་གཞི་གྲངས་མཛོད་མ་འདྲ་བ་རྣམས་ཀྱི་བར་ཕན་ཚུན་སྦྲེལ་ནས་གཅིག་ལ་གཅིག་བརྟེན་གྱི་འབྲེལ་བ་ཚད་ལྡན་ཞིག་འཛུགས་ཐུབ་པ་ཡིན།

BUDA དྲ་ཐོག་དཔེ་མཛོད་ཀྱི་ Linked Open Data ཡི་དཔེ་རྣམ་ནི་དམིགས་བསལ་གྱིས་ནང་པའི་གཅེས་ནོར་དུ་གྱུར་བའི་ཆོས་གཞུང་རྣམས་འགྲེལ་བརྗོད་བྱ་ཆེད་གསར་བཟོ་བྱས་ཤིང་། འབྲེལ་ཡོད་ཀྱི་མི་སྣ་སུ་ཡིན་དང་ཅི་ཞིག་ཡིན་པ། དུས་ཚོད། ས་གནས་དང་བཅས་གསལ་བརྗོད་བྱེད་པ་ཡིན། འདིའི་ནང་ཆེད་མཁས་པ་ཚོར་ཆེད་འདེམ་བྱས་པའི་དཔེ་དེབ་གཏོས་ཆེ་བ་ཞིག་ཀྱང་འདུས་ཡོད། དེ་དག་ཀྱང་སྐད་ཡིག་སྣ་ཚོགས་ནང་ནས་ངོས་འཛིན་བྱེད་ཐུབ། ནམ་རྒྱུན་ལྟར་དྲ་ཐག་སྤྱད་དེ་དྲ་ཚིགས་ནང་གི་ཤོག་ངོས་ཕན་ཚུན་མཐུད་པའི་བྱེད་ཐབས་དེ་མི་དགོས་པར་ Linked Open Data ཡིས་རྒྱུ་ཆ་དེ་རྣམས་བུདྡྷ་དྲ་ཐོག་དཔེ་མཛོད་རང་གི་ནང་ཕན་ཚུན་སྦྲེལ་མཐུད་བྱེད་ཐུབ་པ་ཡིན། དེའི་ཕྱིར་བུདྡྷ་དྲ་ཐོག་དཔེ་མཛོད་ནི་རྒྱུ་ཆ་འམ་དཔེ་དེབ་དེ་དག་གི་ལྟེ་གནས་སུ་གྱུར་ཡོད། འདི་བརྒྱུད་ནས་ནང་ཆོས་དོན་གཉེར་ཅན་གང་ན་སུ་བཞུགས་པ་ཐམས་ཅད་ཀྱིས་བུདྡྷ་དྲ་ཐོག་དཔེ་མཛོད་འདི་ལོངས་སུ་སྤྱོད་ཐུབ་པ་ཡིན། དེ་བས་ན་བུདྡྷ་དྲ་ཐོག་དཔེ་མཛོད་འདི་ནི་རྒྱ་ཆེའི་ནང་ཆོས་ལ་དོན་གཉེར་ཆེ་བའི་ཆེད་མཁས་པ་རྣམས་མཉམ་འཛོམས་བྱེད་སའི་སྤྱི་ཚོགསཤིག་ཏུ་གྱུར་ཡོད།

BDRC ཡི་ལོ་རྒྱུས་སྙིང་བསྡུས།

BDRC འམ་ནང་བསྟན་དཔེ་ཚོགས་ལྟེ་གནས་ནི་ནང་པའི་ཆོས་དཔེ་རྣམས་འཚོལ་ཞིབ་དང་ཉར་སྲུང་། བཤེར་འབེབས། ཁྱབ་སྤེལ་བྱེད་པ་བཅས་ལ་འབད་བརྩོན་བྱེད་བཞིན་པའི་ཁེ་མེད་ཚོགས་པ་ཞིག་ཡིན། BDRC ནི་སྐུ་ཞབས་རྗན་སི་མེ་ཐི་(1936-2010) མཆོག་གིས་ཕྱི་ལོ་༡༩༩༩ ལོར་ཐོག་མར་TBRC བོད་བརྒྱུད་ནང་བསྟན་དཔེ་ཚོགས་ལྟེ་གནས་ཀྱི་ངོ་བོར་གསར་ཏུ་བཙུགས། དེ་ནས་བཟུང་ཚོགས་པ་འདིས་མདོ་སྡེ་རྣམས་དང་འགྲེལ་བ་ཁག ཉམས་ལེན་གྱི་དཔེ་དེབ་ཁག དེ་བཞིན་ཆོ་གའི་དཔེ་དེབ་སོགས་འདུས་པའི་ཁྱོན་བསྡོམས་ནང་ཆོས་ཀྱི་གཞུང་ཤོག་ངོས་ས་ཡ་༢༠ ལྷག་ཨང་བསྒྱུར་བྱས་པ་དང་དཀར་ཆག་བསྒྲིགས་ཏེ་ཉར་སྲུང་བྱས་ཡོད། BDRC ཡིས་བརྩམས་ཆོས་དེ་རྣམས་བསྐལ་བ་ཇི་སྲིད་གནས་པ་དེ་སྲིད་དུ་གནས་ཕྱིར་གསོག་ཉར་བྱས་ཡོད། དེ་བཞིན་དཔེ་ཚོགས་དེ་དག་བརྟན་གནས་དང་གང་ན་སུ་བཞུགས་པ་ཐམས་ཅད་ཀྱིས་འབྱུང་འགྱུར་ཐུན་མོང་གིས་ལོངས་སུ་སྤྱོད་ཐུབ་པར་འབད་འབུངས་བྱེད་བཞིན་པ་བཅས་སོ།།

སྐུ་ཞབས་རྗན་སི་མེ་ཐི་མཆོག་གི་ཕུག་རྒྱང་རིང་འཆར་གཞི་སློབ་སྟོན་ལ་བརྟེན་ནས་ BDRC ཚོགས་པས་རྩ་ཆེ་བའི་བོད་ཀྱི་རྩོམ་རིག་བརྩམས་ཆོས་རྣམས་འཇིག་ཉེན་ལས་བསྐྱབས་ཏེ་ཚད་མཐོ་བའི་ཉར་ཚགས་བྱེད་ཐུབ་ཡོད། དེང་གི་སྐབས་འདིར་བོད་རིག་པ་སློབ་གཉེར་བྱེད་མཁན་ཞིག་ཡིན་ན་BDRC ཡི་དྲ་ཐོག་དཔེ་མཛོད་འདི་ནི་གལ་འགངས་ཧ་ཅང་ཆེ་བའི་མཐུན་རྐྱེན་ཞིག་ཡིན་ཏེ། རྒྱལ་སྤྱི་རང་བཞིན་གྱི་གཙུག་ལག་ཆེད་མཁས་པ་རྣམས་དང་ནང་པའི་ཚོགས་སྡེ་ཁག ལོ་ཙཱ་བ། དཔེ་བསྐྲུན་པ། དེ་བཞིན་དོན་གཉེར་ཅན་གྱི་མང་ཚོགས་རྣམས་ཀྱང་དཔེ་མཛོད་འདི་ལ་བརྟེན་ཡོད།

ཕྱི་ལོ་༢༠༡༥ ལོར་ངེད་ཚོགས་པའི་སློབ་སྟོན་ལྷན་ཚོགས་དང་མཉམ་འབྲེལ་ཚོགས་སྡེ་ཁག དེ་བཞིན་ནང་བསྟན་སློབ་གཉེར་ཐད་ཕྱག་ལས་གནང་མཁན་གྱི་ལས་གྲོགས་བཅས་ཀྱི་རེ་བ་ལྟར། TBRC བོད་བརྒྱུད་ནང་བསྟན་དཔེ་ཚོགས་ལྟེ་གནས་ཀྱི་ལས་གཞི་རྒྱ་བསྐྱེད་བྱ་རྒྱུའི་ཐག་གཅོད་བྱས། ཕྱི་ལོ་༢༠༡༦ ལོར་བོད་བརྒྱུད་ནང་བསྟན་དཔེ་ཚོགས་ལྟེ་གནས་ཀྱི་ལས་གཞི་དེ་གཞུང་འབྲེལ་རང་བཞིན་གྱིས་BDRC འམ་ནང་བསྟན་དཔེ་ཚོགས་ལྟེ་གནས་ཀྱི་ངོ་བོར་བསྒྱུར། ད་དུང་སྐུ་ཞབས་རྗན་སི་མེ་ཐི་མཆོག་གིས་དབུ་འཛུགས་བྱས་པའི་ལས་གཞི་དེ། ནང་བསྟན་དཔེ་ཚོགས་ལྟེ་གནས་ཀྱིས་གཞན་དབང་སྟབས་བསྟུན་གྱི་སྤྱི་ཚོགས་དང་ཁོར་ཡུག་གི་གནད་དོན་གང་ཅིའི་རྒྱུ་རྐྱེན་ལ་བརྟེན་ནས་ཉེན་གནས་སུ་གྱར་བའི་རིས་མེད་ནང་བསྟན་སྤྱིའི་གསུང་རབ་རྣམས་ཉར་སྲུངདང་ཁྱབ་སྤེལ་བྱེད་པའི་ལས་འགན་བསྒྲུབ་དགོས་བྱུང་། དུས་དེ་ནས་ད་ལྟའི་བར་BDRC ནང་བསྟན་དཔེ་ཚོགས་ལྟེ་གནས་ཀྱིས་ཆེད་ལས་རང་བཞིན་གྱི་འཕྲུལ་རིགས་ལག་རྩལ་བཀོལ་སྤྱོད་དང་ཆེད་མཁས་རང་བཞིན་གྱི་ཕྱག་དཔེ་རྣམས་གཅེས་ཉཪ་ཁྱབ་སྤེལ་གྱི་ལས་གཞི་དེ་མི་ལོ་ཉི་ཤུ་ལྷག་གི་རིང་ཛ་དྲག་རང་བཞིན་གྱིས་བསྒྲུབས་དང་སྒྲུབ་མུས་སུ་མཆིས།
